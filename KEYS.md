# PURA — Keys to Add After Build

## Web3Forms (email backend)
1. Go to https://web3forms.com
2. Enter viveapr@gmail.com and click Get Access Key
3. Copy the key
4. In every HTML file, replace: WEB3FORMS_ACCESS_KEY with your actual key

## Anthropic API (AI responses)
1. Go to https://console.anthropic.com
2. API Keys → Create Key
3. In every HTML file, replace: ANTHROPIC_API_KEY with your actual key

## Supabase (data storage)
Project URL: https://oljhhgodblludybhndgj.supabase.co
1. Go to Supabase dashboard → Settings → API
2. Copy the anon/public key
3. In every HTML file, replace: SUPABASE_ANON_KEY with your actual key
